# Joy (조이) - AI Desktop Assistant 🤖✨

**Joy**는 100% 로컬 환경에서 동작하는 귀여운 AI 데스크탑 비서입니다.  
내 컴퓨터 화면을 이해하고, 작업을 도와주며, 절대 외부로 데이터를 유출하지 않습니다.

![Joy Preview](assets/joy_preview.png)

## ✨ 주요 기능 (Key Features)

- **👀 시각 능력 (Vision)**: 화면을 직접 보고 질문에 대답합니다. ("이 에러 뭐야?", "이거 해석해줘")
- **✅ 자동 시각 모드 (Auto-Vision)**: 대화할 때마다 자동으로 화면을 인식하여 문맥을 파악합니다.
- **🧠 100% 로컬 AI**: `Ollama`를 사용하여 내 컴퓨터 안에서만 생각합니다. (인터넷 불필요, 보안 걱정 ZERO)
- **💬 한국어 최적화**: 어떤 질문에도 친절하고 정확한 한국어로 대답합니다.
- **🎨 투명한 UI**: 배경 없는 깔끔한 캐릭터가 화면 한구석에서 언제나 대기합니다.

## 🛠️ 설치 및 실행 (Setup)

1. **Ollama 설치**: [Ollama 공식 홈페이지](https://ollama.com/)에서 다운로드 후 설치합니다.
2. **모델 다운로드**: 터미널에서 다음 명령어를 실행합니다.
    ```bash
    ollama pull llama3    # 텍스트 대화용
    ollama pull llava     # 화면 인식용
    ```
3. **실행**:
    ```bash
    setup.bat
    ```

## 🎮 사용 방법 (Controls)

- **좌클릭 드래그**: Joy를 원하는 위치로 옮길 수 있습니다.
- **우클릭 메뉴**:
    - `👀 현재 화면 보고 대답하기`: 즉시 화면을 캡처하고 질문을 기다립니다.
    - `✅ 대화할 때마다 화면 보기`: 켜두면 항상 화면을 보며 대화합니다.
    - `원위치로 이동`: Joy가 화면 밖으로 나갔을 때 돌아오게 합니다.
    - `종료`: Joy를 재웁니다. 😴

## 🔒 보안 및 프라이버시 (Privacy)

Joy는 **모든 데이터(화면 캡처, 대화 내용)를 사용자 컴퓨터의 메모리(RAM)에서만 처리**합니다.  
처리 직후 데이터는 파기되며, 외부 서버로 전송되는 일은 절대 없습니다. 안심하고 회사 업무나 개인 작업에 활용하세요!
